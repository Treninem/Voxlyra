# Обновление VoxLyra v1.11.0 → v1.11.1

1. Сохраните свой файл `.env`.
2. Сохраните `data/voxlyra.sqlite3`.
3. Сохраните всю папку `storage/`.
4. Распакуйте архив обновления поверх установленной v1.11.0 с заменой файлов.
5. Выполните Redeploy.

## После запуска

- Мягкая миграция откроет все главы книг с ценой `0 Stars`.
- Старые цены глав сохранятся только как неактивный черновик.
- Платёжные пакеты текстовых глав бесплатных книг отключатся.
- Ранее принятые документы останутся принятыми и повторно не отправятся.
- Telegram получит новую версию CSS/JavaScript и не должен использовать старую карточку покупки.

Не заменяйте рабочий `.env`, базу и папку `storage` файлами из архива.
