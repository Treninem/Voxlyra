# Обновление VoxLyra v1.11.7 → v1.11.8

1. Сохраните `.env`, `data/voxlyra.sqlite3` и всю папку `storage`.
2. Загрузите содержимое папки `voxlyra_bot` из архива обновления в корень GitHub с заменой файлов.
3. Не удаляйте существующую базу, книги, обложки, аудио, комиксы и юридические документы.
4. Сделайте commit и Redeploy в Bothost.
5. При первом запуске дождитесь завершения автоматической миграции базы.
6. Откройте `Управление → Premium` и проверьте процент фонда авторов — по умолчанию 70%.
7. Проведите тестовую оплату, чтение произведения Premium и проверку авторской аналитики.

Обновление добавляет таблицы фондов Premium и начислений авторам. Существующие оплаченные Premium-покупки получают недостающие фонды автоматически. Ручная выдача Premium денежный фонд не создаёт. Юридическая версия изменена на 2026-07-15, поэтому пользователям и авторам может потребоваться повторное принятие обновлённых документов.
