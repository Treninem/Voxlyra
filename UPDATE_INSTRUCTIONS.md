# Обновление VoxLyra v1.11.11 → v1.11.12

1. Сохраните `.env`, `data/voxlyra.sqlite3` и всю папку `storage`.
2. Скопируйте содержимое папки `voxlyra_bot` из архива обновления в корень проекта с заменой файлов.
3. Не удаляйте базу и пользовательские файлы.
4. Выполните Redeploy на Bothost.
5. Полностью закройте Mini App и откройте его заново через бота.
6. В кабинете автора откройте книгу с несколькими сотнями глав.
7. Удалите прежний пост в канале и выполните повторную публикацию: старое сообщение Telegram само не изменится.
8. Проверьте кнопку `Открыть книгу`: она должна сразу запустить Mini App на странице произведения. Синей ссылки в тексте публикации больше быть не должно.

Для надёжной прямой ссылки у бота должен быть настроен Main Mini App в BotFather. Сборка использует `BOT_USERNAME` из `.env`, а при запуске дополнительно определяет фактическое имя бота через Telegram.
