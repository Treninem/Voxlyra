# Обновление Voxlyra с v1.8.5 сразу до v1.8.7

Этот архив уже содержит озвучивание из v1.8.6 и служебный доступ проверки из v1.8.7. Отдельно устанавливать v1.8.6 не нужно.

1. Сохраните рабочие `.env`, `data/voxlyra.sqlite3` и всю папку `storage/`.
2. Распакуйте архив обновления поверх текущего проекта v1.8.5.
3. Не заменяйте рабочую базу, обложки, книги, аудиофайлы и кэш озвучивания.
4. Переменную окружения не меняйте. Она должна остаться:

```env
PROJECT_VERSION=v1.8.3-author-cover-display
```

5. Выполните **Redeploy**, а не Restart. Docker должен установить локальные `espeak-ng` и `ffmpeg`.
6. Проверьте обычную платную главу: читатель без покупки должен видеть оплату.
7. Откройте скрытую панель владельца или модератора с правом «Модерация книг».
8. В очереди книг нажмите **«Читать книгу»**. В читалке должна появиться отметка служебного режима.
9. Проверьте текст, выбор голоса и озвучивание платной главы.
10. У администратора без права `mod_books` служебный доступ должен отсутствовать.
11. В скрытом меню владельца должна отображаться сборка `v1.8.7 · служебная проверка платных глав`.

Новых обязательных переменных окружения нет. `PROJECT_VERSION` менять не требуется.
