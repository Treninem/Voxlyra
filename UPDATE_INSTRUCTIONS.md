# Обновление Voxlyra v1.10.2 → v1.10.3

1. Сохраните `.env`.
2. Сохраните `data/voxlyra.sqlite3`.
3. Сохраните папку `storage/`.
4. Распакуйте обновление поверх v1.10.2 с заменой файлов.
5. Выполните Redeploy.

База данных не сбрасывается. Новых обязательных переменных окружения и новых системных пакетов нет. Рейтинги начнут рассчитываться автоматически по уже накопленным покупкам, чтению, прослушиванию, библиотеке и отзывам.
