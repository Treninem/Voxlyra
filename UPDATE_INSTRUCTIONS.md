# Обновление VoxLyra v1.11.8 → v1.11.9

1. Сохраните `.env`, `data/voxlyra.sqlite3` и всю папку `storage`.
2. Загрузите содержимое папки `voxlyra_bot` из архива обновления в корень GitHub с заменой файлов.
3. Не удаляйте базу, книги, обложки, аудио, комиксы и юридические документы.
4. Сделайте commit и Redeploy в Bothost.
5. Полностью закройте Telegram Mini App и заново откройте бота.
6. Создайте тестовый черновик книги и проверьте тип, разделы жанров, теги, аудиторию, предупреждения и подсказку возраста.

Миграция базы не требуется: обновление расширяет программный классификатор и использует существующую таблицу `book_option_values`.
