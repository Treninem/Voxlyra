# Обновление VoxLyra v1.10.5 → v1.11.4

Это накопительное обновление для установленной у вас **v1.10.5**. Оно уже включает изменения v1.11.0–v1.11.3 и новое обновление v1.11.4.

1. Сохраните `.env`, `data/voxlyra.sqlite3` и всю папку `storage`.
2. Распакуйте архив обновления.
3. Загрузите содержимое архива в корень GitHub-репозитория с заменой файлов. В корне должны быть `Dockerfile`, `main.py`, `requirements.txt`, `app/`, `static/`, `templates/`.
4. Не создавайте дополнительную вложенную папку `voxlyra_bot/voxlyra_bot/`.
5. Выполните commit и нажмите Redeploy в Bothost.
6. Не удаляйте существующую базу и пользовательские файлы `storage`. Устаревшие юридические PDF прежней редакции будут удалены автоматически самим `scripts/start.sh`; книги, база, аудио и обложки не затрагиваются.

## После запуска

- Владелец увидит v1.11.4 в защищённом центре управления.
- Для выдачи доступа откройте «Управление → Выдать доступ».
- Введите Telegram ID, `@username`, имя или псевдоним пользователя.
- Для глав допускаются записи `33`, `56-67`, `98 34,36,38`, `1-100`.
- Право сотруднику выдаётся отдельно: «Выдача доступа и Premium».
- Автор выбирает режим Premium в карточке управления книгой. При переходе из бесплатного режима первые три главы остаются ознакомительными, остальные открываются по подписке.
- Vosk не блокирует Docker-сборку: модель подготавливается после запуска, как в v1.11.3.

Номер версии не показывается обычным пользователям.
