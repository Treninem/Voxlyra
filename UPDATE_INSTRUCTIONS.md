# Обновление Вокслиры до v1.8.3

1. Сохраните рабочий `.env`.
2. Сохраните `data/voxlyra.sqlite3`.
3. Обязательно сохраните всё содержимое `storage/`, особенно `storage/covers/`.
4. Скопируйте файлы из архива обновления поверх проекта v1.8.2.
5. Не заменяйте рабочий `.env`, базу, загруженные книги, обложки и аудио.
6. В Bothost установите `PROJECT_VERSION=v1.8.3-author-cover-display`.
7. Выполните **Redeploy**, а не обычный Restart.
8. Откройте Mini App через бота.
9. Перейдите в `Моё → Кабинет автора → Мои книги`.
10. Проверьте, что обложка видна на карточке черновика и в редакторе книги.

Публичный каталог по-прежнему показывает только опубликованные книги. Черновая обложка доступна только владельцу книги через проверенную Telegram-сессию.
