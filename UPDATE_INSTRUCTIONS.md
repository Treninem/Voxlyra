# Обновление Voxlyra v1.10.4 → v1.10.5

1. Сохраните файл `.env`.
2. Сохраните `data/voxlyra.sqlite3`.
3. Сохраните всю папку `storage/`.
4. Распакуйте архив обновления поверх установленной v1.10.4 с заменой файлов.
5. Выполните Redeploy.

## Что произойдёт после запуска

- Структура базы данных не сбрасывается и отдельная миграция не требуется.
- Книги, главы, покупки, комиксы, обложки и загруженные авторами аудиокниги сохраняются.
- Один раз очищается только старый автоматически созданный кэш озвучивания в `storage/tts/`.
- В браузере Telegram удаляется прежняя версия кэша TTS.
- Озвучивание начинает работать короткими фрагментами с предварительной загрузкой продолжения и следующей главы.

## Голосовые движки

Обновление сразу работает с локальным Piper. Новые переменные `.env` необязательны. Для более естественного тембра можно отдельно подключить собственный сервер Qwen или MOSS по инструкции `docs/TTS_PRIVATE_SERVER_CONTRACT_V1_10_5.md`. Тяжёлые модели и их весовые файлы в архив Bothost не входят.

Не переносите `.env` из архива и не заменяйте свою рабочую базу пустым файлом. Оплата цифрового содержимого остаётся только через Telegram Stars. ЮKassa не активируется. Аватары бота и канала не изменяются.
