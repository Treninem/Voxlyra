# Обновление Voxlyra v1.10.3 → v1.10.4

1. Сохраните `.env`.
2. Сохраните `data/voxlyra.sqlite3`.
3. Сохраните всю папку `storage/`.
4. Распакуйте архив обновления поверх v1.10.3 с заменой файлов.
5. Выполните Redeploy.

Изображения уже находятся внутри обновления в оптимизированном WebP-формате и автоматически подключены к Mini App. Отдельно загружать их на хостинг не нужно.

База данных не сбрасывается. Новых обязательных переменных окружения, миграций и системных пакетов нет. Оплата остаётся только через Telegram Stars. ЮKassa не активируется. Аватары бота и канала не изменяются.
