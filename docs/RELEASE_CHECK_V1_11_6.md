# Проверка релиза VoxLyra v1.11.6

Проверить после Redeploy:

1. Кабинет автора на ширинах 320, 360, 390 и 430 px.
2. Логотип не перекрывает «Кабинет автора».
3. Карточка профиля и кнопка «Моя библиотека» полностью видны.
4. Аналитика, достижения и комиссии не имеют горизонтальной прокрутки.
5. Нижняя панель не закрывает последний элемент страницы.
6. В центре владельца отображается v1.11.6.
