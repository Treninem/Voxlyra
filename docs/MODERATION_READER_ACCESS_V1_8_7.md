# Voxlyra v1.8.7 — служебная проверка платных глав

## Кому доступно

- владельцу бота;
- активному сотруднику с правом `mod_books`.

Наличие любого другого административного права не открывает платный текст.

## Что доступно

- чтение опубликованных платных глав без покупки;
- чтение глав книги, находящейся в статусе `review`;
- озвучивание тех же глав локальным TTS;
- переход между главами и переход по номеру;
- сохранение позиции озвучивания.

## Ограничения

- служебный режим не создаёт покупку;
- служебное чтение не попадает в обычный прогресс и рекомендации;
- комментарии и реклама скрываются;
- после отзыва `mod_books` ранее созданная MP3-ссылка перестаёт открываться;
- обычный пользователь и администратор без `mod_books` остаются за платным доступом.

## Журнал

Фиксируются действия:

- `moderation_chapter_read`;
- `moderation_chapter_tts`.
