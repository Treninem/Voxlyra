# VoxLyra v1.11.9 — адаптивные жанры, теги и предупреждения

## Что изменено

Классификация книги теперь строится последовательно и осмысленно:

1. Автор выбирает форму произведения — роман, веб-сериал, новеллу, сборник, книгу-игру, руководство, справочник и другие варианты.
2. Система показывает наиболее подходящие жанры для выбранной формы, но не скрывает остальные.
3. Сюжетные теги подбираются по форме книги и выбранным жанрам.
4. Аудитория подбирается по возрасту, интересам, формату и привычному стилю чтения.
5. Предупреждения предлагаются по жанрам и тегам, но никогда не выбираются автоматически.
6. По предупреждениям показывается предварительная подсказка возрастной маркировки. Окончательный выбор остаётся за автором.

## Размер классификатора

- 27 типов книги;
- 231 жанр;
- 221 сюжетный тег и особенность;
- 48 вариантов аудитории;
- 80 предупреждений по содержанию.

## Удобство выбора

Длинные списки больше не заставляют автора перелистывать десятки страниц подряд. У каждого набора есть:

- раздел «Подходящее»;
- тематические разделы;
- раздел «Выбрано» для проверки отмеченных пунктов;
- полный алфавитный список;
- счётчик выбранных значений;
- постраничная навигация внутри выбранного раздела.

Тип книги также разбит на страницы по 9 вариантов.

## Логика предупреждений

Пункт «Без особых предупреждений» взаимоисключающий:

- при его выборе остальные предупреждения снимаются;
- при выборе любого конкретного предупреждения пункт «Без особых предупреждений» снимается;
- если автор ничего не отметил и нажал «Готово», сохраняется явное значение «Без особых предупреждений».

Предлагаемые предупреждения являются только подсказкой. Система не утверждает, что чувствительная тема присутствует в книге.

## Совместимость

Схема базы данных не менялась. Старые коды жанров, тегов, типов и предупреждений сохранены, поэтому существующие книги и рекомендации не повреждаются. Новые значения сохраняются в той же таблице `book_option_values`.
