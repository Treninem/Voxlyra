# Карта интерфейса Вокслиры

## Обычный читатель

Видит:

```text
📚 Читать
🎧 Слушать
⭐ Моё
➕ Ещё
```

Внутри `➕ Ещё`:

```text
✍️ Автору
💎 Бонусы
🛟 Поддержка
📜 Правила
⚙️ Настройки
```

Не видит:

```text
👑 Управление
🛡 Модерация
служебную версию
финансовые настройки
список администраторов
```

## Автор

После регистрации видит:

```text
📚 Мои книги
➕ Добавить книгу
🎧 Аудиокниги
💰 Доход
📢 Продвижение
👤 Профиль
📜 Правила авторов
```

Автор пишет руками только:

```text
название
описание
текст главы
имя диктора
свободные реквизиты выплаты
```

Автор выбирает галочками:

```text
тип книги
язык
жанры
сюжетные теги
аудиторию
предупреждения
возраст
статус
скачивание
тип продажи
```

## Модератор

Модератор видит только выданные владельцем права.

Примеры прав:

```text
📚 Модерация книг
💬 Модерация комментариев
🧾 Работа с жалобами
↩️ Возвраты
📢 Реклама
📣 Канал
📊 Статистика
```

Недоступные разделы не показываются.

## Владелец

Только владелец из `OWNER_IDS` видит:

```text
👑 Управление
```

Внутри:

```text
👥 Администрация
📚 Книги
✍️ Авторы
👤 Пользователи
💰 Финансы
📢 Канал
🧾 Жалобы
📊 Статистика
⚙️ Настройки
🛡 Безопасность
🧩 Система
```

Только владелец может:

```text
выдавать права администрации
убирать права
менять комиссии
управлять выплатами
смотреть служебную версию
менять рекламу в читалке
блокировать пользователей
блокировать книги
смотреть системные проверки
```
