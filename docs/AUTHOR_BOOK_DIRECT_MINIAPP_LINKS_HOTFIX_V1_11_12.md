# VoxLyra v1.11.12 — кабинет больших книг и прямые ссылки Mini App

## Кабинет автора

Раньше `/api/author/book/{id}` отдавал полный текст сразу всех глав. Для книги на сотни глав ответ становился слишком тяжёлым, из-за чего Mini App показывал «Не удалось выполнить действие».

Теперь основной ответ содержит только сведения о главах без поля `text`. Список разбит в интерфейсе по 100 глав. Полный текст загружается отдельным защищённым запросом только после нажатия на конкретную главу.

## Публикации в канале

Кнопка под публикацией ведёт на основное Mini App по адресу вида:

`https://t.me/VoxlyraBot?startapp=book_42`

Параметр `book_42` открывает нужную книгу. При внутреннем переходе сохраняются параметры запуска Telegram, поэтому авторизация Mini App не теряется.

Синяя ссылка удалена из подписи публикации. Адрес остаётся только в кнопке `Открыть книгу`.

Старые веб-ссылки `/book/{id}`, открытые вне Telegram, перенаправляют на соответствующий прямой запуск Mini App.

## Совместимость

База данных, книги, главы, обложки, аудио, комиксы, покупки и Premium не сбрасываются. Новая миграция базы не требуется.
