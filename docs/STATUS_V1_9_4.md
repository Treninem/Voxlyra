# Статус Voxlyra v1.9.4

## Завершено

- скрытая панель владельца для Stars, ЮKassa и выплат;
- независимое включение и выключение платёжных контуров;
- ввод тестового и боевого provider token BotFather;
- ввод ShopID, секретного ключа магазина и ключей выплат;
- зашифрованное хранение секретов;
- проверка ключей магазина и шлюза выплат;
- запрет подмены Stars ЮKassa для цифрового контента внутри Telegram;
- блокировка создания новых Stars-счетов при отключении;
- защита текстовой и графической читалки при запрете скачивания автором;
- персональный водяной знак;
- скачивание текста только при разрешении автора;
- отключение постоянного кэша комиксов при запрете скачивания;
- документация по значениям BotFather и ЮKassa.

## Ограничения

- системные скриншоты и съёмку экрана невозможно гарантированно запретить в браузерном Mini App;
- рублёвая продажа цифровых книг не выводится внутри Telegram;
- боевые ключи и реальные договоры не входят в исходный код;
- внешняя рублёвая витрина должна запускаться отдельно после юридической и кассовой проверки.

## Совместимость

Обновление ставится поверх v1.9.3. База и старые покупки сохраняются. Аватары бота и канала не меняются.

- 67/67 проверок развёртывания;
- Python и JavaScript без синтаксических ошибок.
