# VoxLyra v1.7.1 — новые аватарки

Что изменено:
- Аватар бота пересобран в чистом премиальном стиле: крупная золотая V, открытая книга, фиолетовое свечение, безопасные отступы для круглой обрезки Telegram.
- Аватар канала пересобран отдельно: микрофон, золотое перо, звуковая волна, тёмно-синий фон, безопасные отступы.
- Внутри проекта заменены файлы `static/img/bot_avatar.png` и `static/img/channel_avatar.png`.

Правило на будущее:
- Не использовать мелкие элементы по краям.
- Не добавлять текст в аватар.
- Центральный объект должен занимать 55–65% площади, чтобы Telegram не срезал важные детали.
