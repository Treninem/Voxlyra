# Вокслира v1.7.8 — премиальная главная и карточки книг

Версия: `v1.7.8-premium-home-stage6`.

## Что изменено

- Главная Mini App перестроена как книжная витрина с крупным первым экраном.
- Добавлены живые подборки: новинки, реальные популярные книги, аудиоверсии и бесплатные главы.
- Блок «Сейчас читают» выводится только при наличии реальных покупок, отзывов или рейтинга.
- Подборки и фильтры, для которых нет подходящих книг, скрываются.
- Карточки книг показывают реальные жанры, возраст, рейтинг, количество глав, аудио, бесплатные главы и цену.
- Кнопка «Читать» ведёт сразу в первую опубликованную главу, когда она существует.
- Кнопка «Слушать» появляется только при наличии опубликованной аудиоглавы.
- Кнопка «В библиотеку» сохраняет книгу через рабочий API.
- Нижняя навигация получила единый набор векторных иконок.
- Добавлена адаптация для телефона, планшета, светлой и сепия-темы.
- Учтена системная настройка уменьшения анимации.

## Главное правило

Интерфейс не имитирует активность. Не показываются пустые рекомендации, несуществующее аудио и действия без рабочего сценария.

## Проверка

- 37 автоматических тестов.
- Python и JavaScript проходят синтаксическую проверку.
- Каталог, аудио и страница книги отдают успешный ответ.
- На ширине 390 и 760 пикселей нет горизонтального переполнения страницы.
- Проверены карточки с бесплатной книгой, платной книгой, жанрами, рейтингом и аудио.
