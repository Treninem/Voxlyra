# Контракт собственного TTS-сервера VoxLyra v1.10.5

Основной бот может находиться на Bothost, а тяжёлая модель — на отдельном закрытом сервере с GPU. Бот не требует публичного API сторонней компании: он обращается только к указанному владельцем адресу.

## Проверка состояния

```http
GET /health
Authorization: Bearer <TTS_REMOTE_TOKEN>
```

Успешный ответ должен иметь код 2xx. Тело может быть произвольным JSON, например:

```json
{"ok": true, "model": "qwen", "warmed": true}
```

## Создание фрагмента

```http
POST /synthesize
Content-Type: application/json
Authorization: Bearer <TTS_REMOTE_TOKEN>
```

Пример запроса:

```json
{
  "text": "— Я вернусь до рассвета, — тихо сказал он.",
  "voice": "dmitri",
  "style": "natural",
  "format": "mp3",
  "sample_rate": 24000,
  "stream": false,
  "segment_kind": "dialogue",
  "quality_attempt": 0,
  "stable_narration": true
}
```

Сервер должен вернуть необработанные байты WAV, FLAC, OGG или MP3 с кодом 2xx. Основной бот самостоятельно приводит звук к моно MP3, выравнивает технические параметры и выполняет контроль качества.

## Сопоставление голосов

Основной бот передаёт два стабильных кода:

- `irina` — женский профиль;
- `dmitri` — мужской профиль.

Собственный сервер может сопоставить их со своими голосами или с разрешёнными образцами дикторов. Нельзя использовать голос реального человека без его разрешения.

## Подача

- `natural` — основное литературное чтение;
- `calm` — спокойная подача;
- `expressive` — умеренно выразительная подача без крика и переигрывания.

При `quality_attempt > 0` серверу рекомендуется уменьшить случайность, эмоциональность и вариативность темпа.

## Тайм-ауты и резерв

Если первый фрагмент не получен за `TTS_REMOTE_FIRST_TIMEOUT_SECONDS`, бот переходит к следующему движку. После сетевой ошибки адрес временно блокируется на `TTS_REMOTE_COOLDOWN_SECONDS`, чтобы каждая фраза не ждала один и тот же недоступный сервер.

## Быстрая проверка

После настройки адреса можно выполнить:

```bash
python scripts/tts_provider_check.py --url https://tts.example.com --token TOKEN --voice irina
```

Скрипт проверит `/health`, запросит русский фрагмент, сохранит временный файл и прогонит его через тот же контроль качества, который используется в боте.
