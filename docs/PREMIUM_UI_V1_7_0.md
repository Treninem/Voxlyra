# Voxlyra v1.7.1 — Premium UI

Цель обновления: привести Mini App к премиальному виду, убрать ощущение заглушек, улучшить чтение, аудио, настройки и витрину.

## Что изменено

- Новая главная витрина с баннером, преимуществами и красивыми карточками книг.
- Улучшенный раздел аудиокниг: показывает только реальные аудиоверсии и аккуратное пустое состояние.
- Улучшенная страница книги: обложка, описание, теги, главы, аудиоглавы, отзывы.
- Читалка получила прогресс-бар, фокус-режим, настройки шрифта и скрытие рекомендаций.
- Добавлен раздел «Моё» для личной библиотеки и будущих покупок/закладок.
- Настройки расширены: авто/тёмная/светлая/сепия, размер текста, интервал, ширина читалки, контраст, фокус-режим, рекомендации, аудиоскорость и перемотка.
- Нижняя навигация стала полноценной: Главная, Книги, Аудио, Моё, Стиль.
- Аватарки пересобраны в безопасной центральной зоне без обрезанных краёв.

## Рабочие настройки Bothost

```env
PORT=3000
RUN_WEBAPP=true
WEBAPP_URL=https://voxlyra.bothost.tech
PROJECT_VERSION=v1.7.1-premium-avatars
```

В панели Bothost:

```text
Главный файл: main.py
Использовать собственный Dockerfile: включено
Использовать домен: включено
Порт веб-приложения: 3000
Кастомный домен: voxlyra.bothost.tech
```

## Идеи для следующих этапов

1. «Продолжить чтение» на главной — последняя открытая книга и глава.
2. «Автор недели» и «Книга недели» с автопостом в канал.
3. Книжные коллекции: тёмное фэнтези, романтика, короткие истории, аудио на вечер.
4. Умные бейджи: «есть аудио», «завершена», «бесплатные главы», «новая глава».
5. Таймер сна в аудиоплеере.
6. Скачивание разрешённых книг в EPUB/FB2/PDF из Mini App.
7. Лента автора: новости книги, планы глав, благодарности читателям.
8. Улучшенная статистика автора: дочитывания, прослушивания, конверсия покупки.
9. «Черновой предпросмотр» книги перед отправкой на модерацию.
10. «Песочница» для владельца — тестовые книги и тестовые оплаты без показа пользователям.
