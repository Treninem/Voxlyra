# Как поставить аватар бота и канала

## Аватар бота

Файл в проекте:

```text
static/img/bot_avatar.png
```

Отдельный файл в архиве сборки:

```text
voxlyra_bot_avatar_premium_v1.4.0.png
```

Порядок:

1. Открой Telegram.
2. Найди `@BotFather`.
3. Нажми меню или отправь `/mybots`.
4. Выбери бота Вокслира.
5. Нажми `Edit Bot`.
6. Нажми `Edit Botpic`.
7. Отправь файл `voxlyra_bot_avatar_premium_v1.4.0.png`.

## Аватар канала

Файл в проекте:

```text
static/img/channel_avatar.png
```

Отдельный файл в архиве сборки:

```text
voxlyra_channel_avatar_premium_v1.4.0.png
```

Порядок:

1. Открой Telegram-канал.
2. Нажми на название канала.
3. Нажми карандаш / изменить.
4. Нажми на аватар.
5. Выбери файл `voxlyra_channel_avatar_premium_v1.4.0.png`.
6. Сохрани.

## Почему аватары без мелкого текста

Telegram сильно уменьшает аватарки в списках чатов. Поэтому основной знак должен читаться в маленьком размере: буква, книга, свет, замок/мир, без длинных надписей.
