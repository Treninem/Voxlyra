# Исправление обложек v1.8.2

Версия: `v1.8.2-cover-fix`.

## Причина

При добавлении книги Telegram-бот сохранял `cover_file_id`, но не скачивал изображение. Поле `cover_path`, которое использует Mini App, оставалось пустым.

## Исправление

- добавлено скачивание Telegram-фотографии в `storage/covers/<book_id>.<ext>`;
- путь записывается в `books.cover_path`;
- при запуске восстанавливаются старые обложки с `cover_file_id` и пустым `cover_path`;
- сохранение выполняется атомарно через временный файл;
- ошибки восстановления записываются только в журнал и не останавливают платформу.

## Проверка

- 51 тест успешно;
- 82/82 статических кнопки имеют обработчики;
- проверка развёртывания 21/21;
- добавлены тесты новой загрузки и восстановления старых обложек.
