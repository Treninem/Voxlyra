# Статус Voxlyra v1.9.3

## Готово

- отдельные PDF-документы для читателей и авторов;
- последовательное принятие документов на первом запуске;
- отдельная кнопка для согласия на обработку персональных данных;
- фиксация редакции, SHA-256, источника, даты и Telegram message ID;
- публичные страницы документов и загрузка PDF;
- блокировка цифровой покупки при непринятых актуальных документах;
- отдельные договор и согласие для автора;
- комиссия 20% из окончательной цены читателя;
- калькулятор: чистые 10 ₽ автору → цена 12,50 ₽;
- договорный резерв 14 дней;
- отдельный рублёвый реестр, не смешанный с Telegram Stars;
- зашифрованный телефон СБП;
- платёжный профиль автора с повторной проверкой после изменения;
- очередь проверки платёжных профилей в панели управления;
- получение актуального списка участников СБП через `/v3/sbp_banks`;
- заявка автора на вывод всей доступной суммы;
- очередь рублёвых выплат и ручной запуск выплаты владельцем;
- API-подготовка выплаты ЮKassa через СБП с ключом идемпотентности;
- возврат зарезервированного баланса автору при локальной ошибке или отмене выплаты;
- 108 автоматических тестов;
- 84/84 статические кнопки имеют обработчики;
- 63/63 проверки развёртывания при заполненных обязательных значениях.

## Что намеренно не активировано

- рублёвый checkout цифрового контента внутри Telegram;
- реальная отправка выплат без договора и ключей ЮKassa;
- внешний рублёвый магазин;
- автоматическая проверка статуса самозанятого;
- фискализация и отправка кассовых чеков;
- проверка получателя выплаты через персональные данные ЮKassa;
- webhook и автоматическая сверка финального статуса выплаты.

## Перед реальным запуском

Нужно заполнить реквизиты владельца, выбрать фактическую договорную и налоговую модель, заключить договоры с ЮKassa, получить отдельные данные шлюза выплат, создать постоянный `DATA_ENCRYPTION_KEY`, проверить документы у российского юриста и бухгалтера и провести тестовые операции.

Техническая основа юридического контура: **100%**.
Реальное подключение ЮKassa: **не включено**, так как зависит от реквизитов, договора, проверки и ключей владельца.
