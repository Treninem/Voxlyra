# Текущая точка VoxLyra — v1.11.8

База для обновления: v1.11.7. Финальная сборка: v1.11.8 `premium_author_revenue_integer_stars`.

## Выполнено

- С каждой оплаченной подписки Premium создаётся отдельный фонд авторов.
- По умолчанию 70% цены направляется авторам: при 99 Stars это 69 Stars, платформе остаётся 30 Stars.
- Фонд распределяется по реальному чтению произведений Premium конкретным подписчиком: открытие главы = 1 балл, дочитывание = ещё 2 балла.
- Повторные одинаковые события не учитываются; чтение автором собственной книги исключено.
- Дробные Stars не используются. Метод наибольших остатков даёт для 50/20/15/10/5 результат 35/14/10/7/3 Stars.
- После окончания оплаченного периода начисления добавляются в существующий `author_ledger`, проходят удержание и обычную процедуру выплаты.
- Ручной, подарочный или бонусный Premium не создаёт необеспеченный денежный фонд.
- В кабинете автора и центре владельца добавлены показатели дохода Premium.
- Владелец настраивает процент фонда от 1% до 95%; изменение действует только для будущих оплат.
- Добавлена фоновая процедура расчёта каждые 6 часов.
- Юридические документы обновлены до версии 2026-07-15.
