# Перенос проекта Voxlyra

Продолжать с внутренней сборки `v1.8.7 — служебная проверка платных глав`.

Важно:
- `PROJECT_VERSION` не менять: `v1.8.3-author-cover-display`.
- Внутренняя версия показывается только владельцу.
- Основной проект готов на 100%, обязательных этапов не осталось.
- При обновлении сохранять `.env`, `data/voxlyra.sqlite3` и всю `storage/`.

Последние изменения:
- владелец и сотрудники с `mod_books` читают и озвучивают платные главы для проверки;
- доступ работает для книг на проверке и опубликованных книг;
- кнопка «Читать книгу» добавлена в очередь модерации;
- служебный режим отмечается и записывается в журнал;
- отзыв права немедленно закрывает ранее выданную MP3-ссылку;
- локальное бесплатное озвучивание текста главы без API;
- четыре русских голоса и скорость 0,75×–2×;
- озвучивание только поля главы без рекламы, кнопок и комментариев;
- сохранение позиции, перемотка и таймер сна;
- автоматический переход и визуальное открытие следующей главы;
- Media Session и управление с экрана блокировки;
- защищённые ссылки для платных глав;
- автоматический кэш в `storage/tts/`.

Ранее закреплено:
- автоматическая предварительная проверка и ручная очередь сомнительных книг;
- уведомления владельцу, администраторам, модераторам и автору;
- единая публикационная цепочка;
- обложки во всех карточках и постах канала;
- защита от копий;
- поиск по части названия и опечаткам;
- повторное продвижение в канал;
- крупная загрузка через Mini App;
- корректный прогресс чтения;
- переход к главе по номеру;
- навигация «Назад»/«Отмена» по смыслу.

Проверка: 70 тестов, 83/83 кнопки, 23/23 проверки развёртывания при заполненном окружении. Перед релизом повторно выполнить `pytest`, `scripts/ui_audit.py` и `scripts/deploy_check.py`.
