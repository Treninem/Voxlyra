# Перенос проекта Voxlyra

Продолжать с внутренней сборки `v1.8.5 — модерация, обложки и читалка`.

Важно:
- `PROJECT_VERSION` не менять: `v1.8.3-author-cover-display`.
- Внутренняя версия показывается только владельцу.
- Основной проект готов на 100%, обязательных этапов не осталось.
- При обновлении сохранять `.env`, `data/voxlyra.sqlite3` и всю `storage/`.

Последние изменения:
- предварительная автоматическая проверка новой книги без автоматических блокировок;
- очередь ручной модерации для сомнительных книг;
- уведомления владельцу и сотрудникам с правом проверки книг;
- повторные напоминания при отсутствии решения;
- уведомление автору с причиной возврата и кнопкой открытия книги;
- восстановление и показ обложек во всех разделах Mini App;
- отправка обложки в публикации канала;
- исправленный процент чтения;
- быстрый переход к главе по введённому номеру.

Ранее закреплено:
- единая публикационная цепочка;
- защита от копий;
- поиск по части названия и опечаткам;
- повторное продвижение в канал;
- крупная загрузка через Mini App;
- навигация «Назад»/«Отмена» по смыслу.

Проверка: 63 теста, 83/83 кнопки, 21/21 деплой.
