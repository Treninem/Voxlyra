# Перенос проекта Voxlyra

Продолжать с внутренней сборки `v1.8.4 — публикация, поиск и навигация`.

Важно:
- `PROJECT_VERSION` не менять: `v1.8.3-author-cover-display`.
- Внутренняя версия показывается только владельцу.
- Основной проект готов на 100%, обязательных этапов не осталось.
- При обновлении сохранять `.env`, `data/voxlyra.sqlite3` и всю `storage/`.

Последние обязательные решения:
- владелец после импорта публикует книгу сразу;
- обычный автор отправляется на модерацию автоматически;
- любая публикация идёт через единый сервис и канал;
- карточка канала содержит прямую ссылку и кнопку;
- владелец повторно публикует бесплатно;
- платное продвижение одной книги — раз в 30 дней;
- крупная загрузка открывает нужную книгу и блок импорта;
- копии проверяются по названию и хэшу;
- поиск Mini App поддерживает части названия и опечатки;
- «Назад» и «Отмена» используются по смыслу во всех многошаговых сценариях.

Проверка: 58 тестов, 83/83 кнопки, 21/21 деплой.
