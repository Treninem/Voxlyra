# Проверка релиза VoxLyra v1.11.1

## Автоматически подтверждено

- Все Python-модули компилируются.
- Все JavaScript-файлы проходят синтаксическую проверку Node.js.
- Все 15 Jinja-шаблонов загружаются без синтаксических ошибок.
- 87 статических кнопок Telegram имеют обработчики.
- 157 обязательных файлов, зависимостей и настроек проходят deploy check.
- 195 тестов успешно, 1 тест штатно пропущен по условию среды.
- Отдельные регрессионные тесты подтверждают:
  - отсутствие покупки бесплатной главы за `0 Stars`;
  - открытие всех глав книги с нулевой ценой при миграции;
  - сохранение согласий после обычного обновления;
  - возможность явно потребовать повторное согласие для существенной редакции;
  - скрытие версии от публичного интерфейса.

## После установки проверить вручную

1. Бесплатную книгу с ранее закрытой главой — текст должен открыться, платёжных элементов быть не должно.
2. Повторный `/start` пользователя, уже принявшего документы — документы повторно не отправляются.
3. Центр управления владельца — там отображается v1.11.1.
4. Обычный пользовательский интерфейс — версия нигде не показывается.
5. Несколько глав подряд в озвучивании и реальную оплату Stars в Telegram.
