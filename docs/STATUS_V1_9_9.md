# Статус Voxlyra v1.9.9

## Гибкие пакеты глав

Автор может создавать для одного произведения любое количество собственных предложений, например:

- 20 любых глав за 99 Stars;
- 50 любых глав за 240 Stars;
- 100 любых глав за 490 Stars.

Пакет не привязан к подряд идущим главам. После оплаты читатель получает указанное количество персональных открытий внутри конкретного произведения и сам выбирает, какие платные главы открыть.

Пример: после покупки пакета на 20 глав читатель может открыть главу 40, затем главы 50–60, потом главы 101–108. Пропущенные главы не расходуют пакет. Повторное открытие уже выбранной главы также ничего не списывает.

## Правила работы

- перед каждым списанием показывается подтверждение;
- одна новая платная глава расходует одно открытие;
- бесплатная, ранее купленная или уже открытая из пакета глава не расходует баланс;
- открытая из пакета глава остаётся доступной постоянно;
- несколько купленных пакетов одной книги расходуются по очереди, начиная с самого раннего;
- пакет может действовать на текстовые главы, графические главы либо на оба вида;
- автор может изменить цену, название и размер нового предложения или скрыть его;
- изменение предложения не меняет уже проданные читателям балансы;
- неиспользованный пакет можно отправить на возврат по действующим правилам;
- после расходования хотя бы одного открытия автоматический возврат всего пакета блокируется;
- все покупки выполняются только через Telegram Stars.

## Интерфейс автора

В карточке произведения автору доступен редактор пакетов. Для каждого пакета задаются:

- название;
- количество открытий;
- общая цена в Stars;
- область действия: текст, графические главы или всё произведение;
- активность предложения.

Интерфейс сразу показывает среднюю цену одной главы, но не ограничивает скидку автора.

## Интерфейс читателя

- предложения видны на странице произведения;
- после покупки остаток показывается в библиотеке;
- у закрытой главы появляется кнопка «Открыть из пакета»;
- списание выполняется только после подтверждения;
- текстовые и графические главы работают одинаково;
- читатель может покупать несколько пакетов одного произведения.

## Готовность комиксов

Модуль графических произведений остаётся готов примерно на **90%**. Завершены этапы 1–4. Остался один крупный этап:

- OCR и поиск по распознанному тексту;
- переводные слои;
- покадровое чтение;
- закладки и комментарии к отдельным страницам;
- уведомления о новых графических главах;
- расширенная статистика чтения.

## Проверка

- 131 автоматический тест успешно;
- 1 необязательный тест пропущен;
- 88/88 статических кнопок имеют обработчики;
- Python и JavaScript проходят синтаксическую проверку;
- проверена миграция базы и выбор непоследовательных глав из пакета.
