# VoxLyra v1.7.2 — редактирование и удаление книг/глав

## Что добавлено

### Книги

Автор может открыть `✍️ Автору → 📚 Мои книги → выбрать книгу` и выполнить действия:

- изменить название;
- изменить описание;
- изменить возрастное ограничение;
- изменить статус книги;
- включить или отключить скачивание;
- изменить цену;
- удалить книгу.

Удаление книги мягкое: книга исчезает из кабинета автора и каталога, но история покупок, финансовые записи и аудит сохраняются.

### Главы

Автор может открыть `✍️ Автору → 📚 Мои книги → книга → ➕ Главы → 📚 Список глав → выбрать главу` и выполнить действия:

- изменить название главы;
- заменить полный текст главы;
- изменить цену главы;
- сделать главу бесплатной;
- удалить главу через подтверждение.

Удаление главы мягкое: глава скрывается из книги, а история покупок сохраняется.

## Важные правила

- Если автор меняет важные данные опубликованной книги, книга может быть возвращена на проверку.
- Финансовая история не удаляется.
- Удаление не стирает аудит действий.
- В каталог не попадают книги со статусом `deleted`.
