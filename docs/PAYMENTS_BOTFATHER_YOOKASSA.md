# ЮKassa через BotFather и Telegram Stars

## Что реально делает подключение в BotFather

BotFather переводит владельца в авторизацию ЮKassa, после выбора магазина выдаёт provider token. Бот передаёт этот токен в `sendInvoice`, затем отвечает на `PreCheckoutQuery` и после `SuccessfulPayment` выдаёт товар или услугу.

## Ограничение Telegram

Для цифровых книг, глав, аудио, подписок и другого цифрового контента внутри Telegram используется только валюта `XTR` (Telegram Stars) и пустой `provider_token`. Provider token ЮKassa оставлен для разрешённых физических товаров/услуг и не применяется к цифровым произведениям.

## Единая цена

Автор задаёт цену в Stars. Платформа автоматически показывает расчётный эквивалент в рублях по курсу, установленному владельцем. Один источник цены исключает расхождения между карточками.

Важно: фактическая цена приобретения Stars у конкретного читателя может отличаться из-за платформы, НДС и иных сборов Telegram/App Store/Google Play. Поэтому рублёвое значение маркируется как расчётный эквивалент, а не гарантированная сумма списания.
