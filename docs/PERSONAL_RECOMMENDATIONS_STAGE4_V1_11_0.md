# VoxLyra v1.11.0 — этап 4: персональная лента «Для вас»

## Назначение

На главной странице Mini App появилась личная подборка произведений. Она не заменяет каталог и публичные топы, а помогает читателю быстрее находить новые книги, аудио и графические истории по его фактическим интересам.

## Какие сигналы учитываются

- книги в библиотеке: «Читаю», «Любимое», «В планах», «Прочитано», «Отложено»;
- глубина чтения текстовых глав;
- прослушивание аудио и TTS;
- чтение комиксов и графических глав;
- оплаченные книги, главы, аудио и пакеты;
- оценки пользователя;
- совпадение жанров, сюжетных тегов и аудитории;
- авторы, которых пользователь уже читает.

Отрицательные сигналы («Отложено», низкая оценка, «Не интересно») уменьшают вероятность показа. Уже открытые, купленные, добавленные в библиотеку и собственные книги автора из ленты новых рекомендаций исключаются.

## Защита от накрутки

Персональная лента не меняет публичные рейтинги. Покупки, просмотры и клики по рекомендации не добавляют месту в топе через сам алгоритм рекомендаций. Общая популярность используется только как небольшой ограниченный дополнительный вес и не может вытеснить точные совпадения по интересам.

## Объяснимость

Каждая карточка показывает понятную причину:

- «Похоже на …»;
- «Ваш интерес: …»;
- «Новая история автора …»;
- «Высоко оценено читателями»;
- «Можно начать бесплатно»;
- «Новинка VoxLyra».

Для нового пользователя выводится честная стартовая подборка, пока у платформы недостаточно личных сигналов.

## Управление читателем

- «В библиотеку» сохраняет произведение;
- «Не интересно» скрывает конкретную рекомендацию;
- «Обновить» перезагружает подборку с учётом новых действий;
- открытия и скрытия записываются только для улучшения личной ленты.

## Техническая реализация

- таблица `recommendation_events`;
- `list_personalized_books()` в `app/db.py`;
- API `/api/recommendations/for-you`;
- API `/api/recommendations/events`;
- клиентская загрузка без раскрытия персональной истории в HTML;
- блок скрыт вне проверенной Telegram Mini App-сессии.
