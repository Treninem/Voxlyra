# Итоговый отчёт проверки VoxLyra v1.11.12

Проверено локально:

- ответ кабинета автора не содержит тексты всех глав;
- конкретная глава доступна автору отдельным защищённым запросом;
- список больших книг имеет постраничный вывод;
- прямая ссылка использует `startapp=book_ID`;
- Telegram query/hash сохраняются при переходе к книге;
- публикация не содержит текстового URL;
- кнопка публикации сохраняет прямой адрес Mini App;
- прежние функции проекта проходят регрессионные тесты.

Живая проверка Telegram и Bothost выполняется после Redeploy.
