# Отображение обложек автора v1.8.3

Версия: `v1.8.3-author-cover-display`.

## Исправлено

- Обложка черновика теперь отображается в Mini App в разделе «Кабинет автора → Мои книги».
- Обложка показывается в заголовке редактора выбранной книги.
- Для неопубликованной книги используется отдельный защищённый маршрут автора.
- Изображение запрашивается с проверенной Telegram Mini App-сессией.
- Чужой автор не может получить обложку черновика.
- Неопубликованная обложка не становится публичной через `/media/cover/{book_id}`.
- Для изображения автора отключено долговременное кэширование, поэтому после замены не остаётся старая картинка.
- При ошибке загрузки остаётся аккуратная буквенная заглушка.

## Проверка

- 53 автоматических теста.
- Отдельный тест доступа к обложке черновика владельцем и посторонним автором.
- 82 из 82 статических кнопок имеют обработчики.
- Проверка деплоя: 21 из 21 при заполненных переменных.
- Python и JavaScript проходят синтаксическую проверку.
