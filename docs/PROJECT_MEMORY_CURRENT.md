- Популярность и рекомендации строятся только по реальным данным.

## Обложки книг

- Бот сохраняет Telegram `cover_file_id`, скачивает изображение в `storage/covers/` и записывает `cover_path`.
- На старте восстанавливаются старые обложки, у которых есть `cover_file_id`, но пустой `cover_path`.
- Папку `storage/covers/` нельзя удалять при обновлениях.
- Опубликованные обложки читатель получает через `/media/cover/{book_id}`.
- Обложка черновика не публикуется открыто.
- Автор получает собственную черновую обложку через защищённый маршрут `/api/author/book/{book_id}/cover`.
- Маршрут проверяет Telegram Mini App-сессию и принадлежность книги автору.
- Обложка показывается в списке «Мои книги» и в редакторе книги.
- При недоступности изображения остаётся буквенная заглушка.

## Уведомления

Пользователь управляет четырьмя переключателями:

- все уведомления;
- новые главы;
- новые аудиоглавы;
- скидки и промокоды.

Настройки хранятся в базе и одинаково меняются из Telegram и Mini App. Рассылка идёт заинтересованным читателям. Автор исключается. Каждое событие имеет защиту от повторной доставки.

## Безопасность

- Владельческие права нельзя выдать администратору.
- Старые записи с владельческими правами фильтруются.
- Выплаты доступны только владельцу.
- Блокировка пользователя действует в Telegram, Mini App и оплате.
- Telegram-сессия проверяется по подписи и времени.
- Платный текст не попадает в HTML до подтверждения доступа.
- Подробная диагностика остаётся только в `👑 Управление → 🧩 Система`.
- Публичные API-документы FastAPI отключены.
- Служебная версия не раскрывается в пользовательском HTML.
