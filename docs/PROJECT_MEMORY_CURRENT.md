# Текущая точка VoxLyra — v1.11.12

База для обновления: v1.11.11. Финальная сборка: v1.11.12
`author_large_book_and_direct_miniapp_links_hotfix`.

## Исправлено

- книги на сотни и тысячи глав снова открываются в кабинете автора;
- основной ответ кабинета больше не передаёт полный текст всех глав;
- список текстовых глав разбит по 100 позиций;
- текст отдельной главы подгружается только при её редактировании;
- кнопка под постом канала ведёт на `https://t.me/<бот>?startapp=book_ID`;
- запуск сразу открывает нужную книгу в Main Mini App;
- параметры Telegram сохраняются при внутреннем переходе, поэтому Mini App не теряет авторизацию;
- старые браузерные ссылки на книги перенаправляются в Telegram;
- синяя ссылка удалена из текста публикации, оставлена только кнопка;
- имя бота определяется при запуске, для старых `.env` сохранён резерв `VoxlyraBot`;
- база, книги, главы, обложки, комиксы, аудио, продажи и Premium сохранены.

## После установки

Выполнить Redeploy, полностью переоткрыть Mini App, проверить книгу с 500+ главами и повторно опубликовать старую карточку в канал. Ранее отправленные сообщения Telegram автоматически не изменятся.
