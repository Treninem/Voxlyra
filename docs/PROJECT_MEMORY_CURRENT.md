# Актуальная память проекта «Вокслира»

Текущая внутренняя сборка: `v1.8.7 — служебная проверка платных глав`.
Рабочую переменную окружения не менять: `PROJECT_VERSION=v1.8.3-author-cover-display`.
Версия сборки показывается только владельцу в скрытом центре управления и диагностике.

## Прогресс

- Основной проект: 100%.
- Озвучивание текста в читалке: 100%.
- Служебная проверка платных глав: 100%.
- Незавершённых обязательных этапов: 0.
- В каждом отчёте указывать процент готовности и количество оставшихся этапов.

## Озвучивание читалки

- Работает локально без платных API, API-ключей и внешних TTS-сервисов.
- Docker устанавливает `espeak-ng` и `ffmpeg`.
- Голоса: Анна, Елена, Алексей и Михаил.
- Скорость: 0,75×–2×.
- Озвучивается только `chapters.text`; реклама, кнопки, комментарии и интерфейс не попадают в звук.
- Есть пауза, перемотка, сохранение позиции, таймер сна и автоматическая следующая глава.
- Текст и номер главы обновляются при автоматическом переходе.
- Media Session даёт управление с экрана блокировки на поддерживаемых устройствах.
- Платные главы защищены Telegram-сессией и подписанной временной ссылкой.
- Кэш хранится в `storage/tts/`, ограничен по сроку и объёму.
- Новых обязательных переменных окружения нет.


## Служебная проверка платных глав

- Владелец имеет доступ к тексту и озвучиванию любой неудалённой главы для проверки.
- Сотрудник получает такой доступ только при действующем праве `mod_books`.
- Для сотрудника доступны опубликованные книги и книги в статусе `review`.
- В очереди проверки есть кнопка «Читать книгу».
- В читалке показывается явная отметка служебного режима.
- Реклама и комментарии скрываются, обычный прогресс чтения не сохраняется.
- Открытие текста и запуск TTS записываются в `audit_logs`.
- Подписанная MP3-ссылка повторно проверяет актуальность права `mod_books`.
- Администратор без `mod_books` и обычный читатель не обходят оплату.

## Публикация и проверка

- Любой путь загрузки книги использует единый сервис завершения: Telegram, Mini App, ручная глава и импорт файла.
- Владелец публикуется сразу, если нет однозначного технического/антиспам-нарушения или неподтверждённой копии.
- Книга обычного автора проходит предварительную проверку и передаётся человеку при сомнениях.
- Уведомления получают владелец и сотрудники с правом `mod_books`.
- Первое напоминание через 6 часов, повторные через 12 часов; после решения напоминания прекращаются.
- При возврате на доработку автор получает причину и кнопку открытия книги.
- Опубликованная книга появляется в каталоге и отправляется в канал с обложкой и ссылкой.

## Обложки и читалка

- Обложки сохраняются локально и восстанавливаются из Telegram.
- Во всех карточках используется `/media/cover/<book_id>` с заглушкой только при реальной ошибке.
- Процент чтения считается только внутри текста главы.
- Есть быстрый переход по введённому номеру главы.

## Ранее закреплённое

- Проверка копий: нормализованное название, автор, близкое совпадение и SHA-256 файла.
- Поиск Mini App: части названия, точное название, похожие варианты и небольшие опечатки.
- Владелец повторно публикует бесплатно; платное продвижение одной книги — не чаще раза в 30 дней.
- В многошаговых сценариях использовать «Назад» или «Отмена» по смыслу.

## Сохранение данных при обновлении

Всегда сохранять `.env`, `data/voxlyra.sqlite3` и всю папку `storage/`, включая `storage/tts/`.
