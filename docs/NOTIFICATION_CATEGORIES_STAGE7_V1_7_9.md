# Вокслира v1.7.9 — категории уведомлений

Версия: `v1.7.9-notification-categories-stage7`.

## Что сделано

- отдельный главный переключатель уведомлений;
- категории новых глав, аудиоглав и скидок;
- сохранение выбора в базе из Telegram и Mini App;
- выбор категории реально учитывается при доставке;
- читатели определяются по библиотеке, прогрессу, покупкам и отзывам;
- автор книги не получает читательскую рассылку о собственной книге;
- защита от повторной доставки одного события;
- новые главы и аудио опубликованной книги становятся доступными через рабочие сценарии автора;
- черновики первой версии книги публикуются вместе с одобрением книги;
- промокоды опубликованной книги уведомляют заинтересованных читателей;
- ошибки Telegram не ломают основное действие.

## Проверка

- 40 автоматических тестов;
- 82 статические Telegram-кнопки имеют обработчики;
- Python и JavaScript проходят синтаксическую проверку;
- проверка перед деплоем проходит 19 из 19 при заполненных обязательных переменных;
- архивы проверяются тестом ZIP и SHA-256.

## Развёртывание

```env
PROJECT_VERSION=v1.7.9-notification-categories-stage7
PORT=3000
RUN_WEBAPP=true
WEBAPP_URL=https://voxlyra.bothost.tech
DATABASE_PATH=data/voxlyra.sqlite3
```

Использовать Redeploy. Рабочие `.env` и базу `data/voxlyra.sqlite3` не заменять тестовыми файлами.
