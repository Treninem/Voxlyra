# Статус Voxlyra v1.9.8

## Завершено

- исправлен возврат из карточки опубликованной книги в боте;
- добавлены «К моим книгам» и «Главное меню» в авторских экранах;
- добавлена постоянная навигация «Назад» и «В меню» во внутренних разделах Mini App;
- подключена штатная кнопка возврата Telegram;
- читалка комикса возвращает на страницу произведения;
- продажа отдельной графической главы за Telegram Stars;
- продажа целого тома за Telegram Stars;
- отдельная цена и бесплатный режим для каждого тома;
- бесплатный предпросмотр заданного числа страниц;
- жалобы на конкретную страницу;
- очередь постраничной модерации;
- одобрение, возврат на проверку и отклонение страниц;
- уведомление автора о проблемной странице;
- блокировка отклонённой страницы, включая ранее выданные ссылки;
- платёжная система остаётся только Telegram Stars;
- два расчётных курса и комиссия из v1.9.6 сохранены;
- аватары бота и канала не изменены.

## Готовность комиксов

Модуль графических произведений готов примерно на **90%**. Завершены этапы 1–4. Остался 1 крупный этап:

- OCR и поиск по распознанному тексту;
- переводные текстовые слои;
- покадровое чтение на телефоне;
- закладки и комментарии к конкретной странице;
- уведомления о новых графических главах;
- расширенная статистика чтения.

## Проверка

- 129 автоматических тестов успешно;
- 1 необязательный тест пропущен из-за отсутствия дополнительной системной библиотеки окружения;
- 88/88 статических кнопок имеют обработчики;
- 74/74 проверок развёртывания;
- Python и JavaScript проходят синтаксическую проверку.
