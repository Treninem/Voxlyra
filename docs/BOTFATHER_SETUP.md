# Настройка бота в BotFather

## 1. Создать бота

1. Откройте `@BotFather`.
2. Напишите `/newbot`.
3. Введите отображаемое имя, например: `Вокслира`.
4. Введите username, например: `voxlyra_reader_bot`.
5. Сохраните токен.

## 2. Установить аватар бота

1. В `@BotFather` нажмите `/mybots`.
2. Выберите своего бота.
3. Нажмите `Edit Bot`.
4. Нажмите `Edit Botpic`.
5. Загрузите файл `voxlyra_bot_avatar_final.png`.

## 3. Описание бота

Короткое описание:

```text
Книги, главы и аудиоистории внутри Telegram.
```

Полное описание:

```text
Вокслира — платформа для чтения и прослушивания историй. Каталог книг, аудиоглавы, закладки, покупки, отзывы и кабинет автора прямо внутри Telegram.
```

## 4. Команды

Пользователям команды почти не нужны, но для правил и поддержки можно добавить:

```text
start - запустить бота
terms - пользовательское соглашение
privacy - персональные данные
refunds - возвраты
copyright - авторские права
support - поддержка
```

## 5. Mini App

После деплоя на Bothost и получения `WEBAPP_URL` можно добавить кнопку Mini App через BotFather:

1. `/mybots`
2. выбрать бота
3. `Bot Settings`
4. `Menu Button`
5. указать адрес `WEBAPP_URL`

Если BotFather попросит ссылку, вставьте публичный адрес Bothost без слеша в конце.
