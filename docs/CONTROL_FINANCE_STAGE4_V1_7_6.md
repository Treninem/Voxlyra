# Вокслира v1.7.6 — этап 4

Версия: `v1.7.6-control-finance-stage4`.

## Мобильный центр управления

Страница `/control` открывается из меню владельца или модератора. Сервер проверяет Telegram Mini App `initData`, определяет владельца и загружает права администратора из базы.

Владелец видит общую сводку и все рабочие очереди. Модератор видит только выданные ему разделы:

- книги на проверке;
- комментарии и отзывы;
- жалобы;
- возвраты;
- выплаты авторам.

Пользователь без прав получает нейтральный экран без внутренних данных.

## Защита платежей

- Одинаковый `telegram_payment_charge_id` обрабатывается один раз.
- Повторное событие возвращает уже существующую покупку и не создаёт вторую запись дохода.
- Сумма успешного платежа сравнивается с ценой цели покупки.
- Идентификатор оплаты нельзя повторно использовать для другой покупки.

## Возвраты

- Запрос создаёт только владелец покупки.
- Покупка должна иметь статус оплаты.
- Учитывается срок возврата из `refund_window_days`.
- Дубли активных запросов блокируются.
- После подтверждения Telegram покупка, доступ и запись дохода обновляются одной транзакцией.

## Выплаты авторам

В `author_ledger` добавлена связь `payout_request_id`. Каждая заявка получает только те строки дохода, которые вошли в её сумму.

Допустимый порядок:

- новая → одобрена → выплачена;
- новая/одобрена → заморожена;
- заморожена → новая после разморозки;
- новая/одобрена/заморожена → отклонена.

Нельзя отметить выплату выполненной до одобрения или при активной заморозке автора.

## Проверка

- Python-компиляция проекта;
- синтаксис JavaScript;
- аудит Telegram-кнопок;
- проверка маршрутов Mini App;
- автоматические тесты платежей, возвратов, выплат и панели управления.

## Навигация Mini App

- Пункт «Стиль» переименован в понятное «Настройки».
- В разделе «Моё» кабинет автора показывается только действующему автору.
- Центр управления показывается только владельцу или модератору с выданными правами.
- Для владельца и модератора подпись входа отличается, обычный читатель этот блок не видит.
