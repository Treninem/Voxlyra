# Quality Fix v1.5.0

Сделано:
- Mini App переработан визуально: ровная сетка, нормальные отступы, адаптация под телефон.

- Убраны пользовательские заглушки из основных разделов.

После обновления в Bothost поставить PROJECT_VERSION=v1.5.0-quality-fix и сделать Redeploy.
