# Идеи для следующих улучшений VoxLyra

## Для читателя
- Лента «продолжить»: последняя книга, последняя глава, последняя аудиоглава.
- Персональные подборки: «мрачное фэнтези», «короткие главы на вечер», «с аудио», «бесплатно сегодня».
- Ежедневная витрина: 5 книг, которые меняются раз в сутки.
- Красивые достижения: первая глава, первая аудиокнига, 7 дней чтения подряд.
- Система «тихого режима»: меньше уведомлений, только новые главы избранных книг.

## Для автора
- Мастер публикации на 7 шагов с прогрессом и пропуском необязательных пунктов.
- Предпросмотр карточки книги перед публикацией.
- Проверка качества описания: слишком короткое, нет жанров, нет обложки, нет бесплатных глав.
- Автоматические рекомендации цены: мягко, без навязывания.
- Авторская аналитика: просмотры, открытия глав, дочитывания, продажи, возвраты.

## Для владельца
- Панель «сегодня»: новые пользователи, покупки, жалобы, книги на проверке, доход.
- Быстрый карантин книги: скрыть книгу, заморозить выплаты, запросить подтверждение прав.
- Полный журнал действий модераторов.
- Чистая диагностика только в скрытой панели владельца, без служебного текста для пользователей.

## Для Mini App
- Скелетон-загрузка вместо пустого экрана.
- Анимация нажатия карточек и кнопок.
- Красивый empty-state для каждой вкладки.
- Карточки с бейджами: «Новинка», «Есть аудио», «Первые главы бесплатно», «Завершена».
- Сепия-тема для долгого чтения.
