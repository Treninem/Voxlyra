# Premium Polish v1.6.0

Что исправлено:

- Mini App приведён к премиальному тёмному стилю VoxLyra: карточки, нижняя навигация, ровные отступы, адаптивность под телефон.
- Разделы каталога, книги, читалки, аудио и настроек открываются как полноценные страницы, без служебных заглушек.
- Аудио-раздел показывает только реальные аудиоверсии. Если аудиоглав нет, выводится понятное пустое состояние.
- Аудиоплеер работает с загруженными аудиофайлами, проверяет доступ, поддерживает скорость, перемотку и сохранение места.
- Настройки работают: тема, размер текста, сброс локальных настроек и переход к аудиокнигам.
- В сценарии добавления книги и автора поддерживается пропуск необязательных шагов, чтобы описание, страну, обложку, цену, диктора и другие данные можно было заполнить позже.
- Аватары заменены на безопасные для круглой обрезки Telegram: весь знак помещён в центральную область, без срезанных краёв и без лишнего текста.

После обновления в Bothost поставьте:

```env
PROJECT_VERSION=v1.6.0-premium-polish
WEBAPP_URL=https://bot-1783015725-8670-treninem.bothost.tech
PORT=8080
RUN_WEBAPP=true
BOT_USERNAME=VoxlyraBot
```

Затем сделайте Redeploy / Пересобрать.
