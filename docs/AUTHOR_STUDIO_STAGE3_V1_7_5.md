# Этап 3 — кабинет автора и крупный импорт

Версия: `v1.7.5-author-studio-large-import-stage3`.

## Готово

- Кабинет автора внутри Mini App по адресу `/author`.
- Сводка по книгам, главам, публикациям и доходу.
- Редактирование книги: название, описание, возраст, состояние, цена и скачивание.
- Безопасное удаление книги с сохранением финансовой истории.
- Добавление, редактирование и безопасное удаление глав.
- Отправка книги на модерацию.
- Загрузка крупных книг частями по 6 МБ.
- Предпросмотр найденных глав до сохранения.
- Импорт TXT, DOCX, FB2, EPUB, PDF и ZIP.
- Собственный лимит 50 МБ удалён.
- Для Telegram-чата крупные файлы перенаправляются в Mini App, потому что облачный Bot API не отдаёт боту большие файлы.
- ZIP защищён лимитом общего распакованного объёма.

## Переменные

```env
MAX_BOOK_UPLOAD_MB=0
MAX_BOOK_UNPACKED_MB=2048
```

`MAX_BOOK_UPLOAD_MB=0` отключает собственный лимит размера. Реальный предел после этого зависит от свободного места и ограничений хостинга.
