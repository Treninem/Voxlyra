# Как установить финальные аватары Вокслиры

## Аватар бота

Используйте файл в корне проекта:

```text
voxlyra_bot_avatar_final.png
```

Та же картинка хранится внутри интерфейса:

```text
static/img/bot_avatar.png
```

Порядок установки:

1. Откройте `@BotFather`.
2. Отправьте `/mybots`.
3. Выберите бота Вокслира.
4. Откройте `Edit Bot` → `Edit Botpic`.
5. Отправьте `voxlyra_bot_avatar_final.png`.

## Аватар канала

Используйте файл в корне проекта:

```text
voxlyra_channel_avatar_final.png
```

Та же картинка хранится внутри интерфейса:

```text
static/img/channel_avatar.png
```

Порядок установки:

1. Откройте канал Вокслиры.
2. Нажмите название канала и кнопку редактирования.
3. Нажмите на текущий аватар.
4. Выберите `voxlyra_channel_avatar_final.png`.
5. Сохраните изменения.

## Почему изображения подходят для Telegram

- квадрат 1024 × 1024 пикселя;
- главный знак находится в центре;
- нет текста и мелких деталей у краёв;
- важные элементы остаются внутри круглой обрезки;
- знак читается в маленьком размере списка чатов.
