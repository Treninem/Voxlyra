# VoxLyra v1.13.12 — установка обновления

## Что исправлено
- кнопка «Изменить интервал» больше не выдаёт «Неизвестная настройка»;
- кнопка «Изменить книг за запуск» больше не выдаёт «Неизвестная настройка»;
- работают настройки пакетной библиотеки и обычных авторов;
- значения сохраняются в базе и отображаются после повторного открытия меню;
- при изменении интервала уже ожидающие записи обеих очередей получают новый ближайший слот;
- пакетная очередь соблюдает заданное количество книг за запуск;
- авторская очередь соблюдает заданное количество книг и круговое распределение по авторам.

## Установка
1. Сохранить текущие `.env`, `data/` и `storage/`.
2. Распаковать архив поверх файлов проекта с заменой.
3. Не удалять существующую базу данных.
4. Перезапустить или заново развернуть приложение на Bothost.

Ручные SQL-команды не требуются. База, `.env` и пользовательское хранилище в архив не включены.

## Границы версии
Зависание импорта после текста «Архив загружается...» в эту сборку не входит. Оно запланировано отдельным hotfix v1.13.12.1.
